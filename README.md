# DiaMenu - Smart Diabetic Management for Filipinos

AI-powered menu planning, pantry tracking, and health analytics tailored for the Filipino context.

## Project Objectives

DiaMenu aims to transform diabetic management from a burden into a seamless lifestyle through four core pillars:

1.  **Clinical Precision:** Enabling stable glycemic control via AI-audited, low-GI meal plans and data-driven blood sugar insights.
2.  **Cultural Relevance:** Integrating authentic Filipino cuisine and local market (palengke) accessibility into medical recommendations.
3.  **Smart Automation:** Reducing cognitive load by automating inventory tracking and proactive grocery "auditing."
4.  **Behavioral Change:** Fostering long-term adherence through education, gamification, and privacy-first caregiver support.

For a detailed breakdown of our goals, see [OBJECTIVES.md](./OBJECTIVES.md).

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
